---
title: 'Client script install guide'
description: 'Learn how to install the client side code on your website'
---

To install the script you need a project so make sure you have created one. 
If you haven't, you can create one [here](https://visitorquery.com/dashboard/projects).

Once you have your project, click on the `Settings icon -> Setup instructions` and add the domain name where you wish to install the script.

Copy the `Public API key` form that window and paste it in the script below.

```html
<script>
  (function() {
    var vqScript = document.createElement('script');
    vqScript.type = 'text/javascript';
    vqScript.async = true;
    vqScript.src = 'https://cdn.visitorquery.com/visitorquery.js';
    vqScript.setAttribute('data-api-key', 'YOUR_PUBLIC_API_KEY');
    var s = document.getElementsByTagName('script')[0];
    s.parentNode.insertBefore(vqScript, s);
  })();
</script>
```
